# Avatar

## Data init

Background color is random auto-generated from theme colors. Initial
data from user name if there is no avatar image is the uppercase of
first 2 letters.

# API

| Props       | Type                                        | default | description                     |
| ----------- | ------------------------------------------- | ------- | ------------------------------- |
| size        | string: "xs", "sm", "md", "lg", "xl", "xxl" | "md"    | size of avatar                  |
| presence    | string: "online", "offline", "busy"         |         | mark of user status             |
| avatarUrl   | string                                      |         | link of avatar image            |
| dataInitial | string                                      | "WM"    | initial data name inside avatar |
| alt         | string                                      |         | alt attribute                   |

_(Note)_ Avatar can be extended with props [space](https://styled-system.com/api#space) from **styled-system**.
