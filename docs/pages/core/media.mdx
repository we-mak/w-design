import { Image, ImageRatio, Picture } from "@w-design/core";

# Image

Image component with built-in lazy loading with `Intersection Observer`.

## Example

Fixed width image

<Image
  data-src="https://source.unsplash.com/random"
  width={320}
  height={500}
  style={{ border: "1px solid teal" }}
/>{" "}

```jsx
<Image
  data-src="https://source.unsplash.com/random"
  width={320}
  height={500}
  style={{ border: "1px solid teal" }}
/>
```

Responsive aspect ratio style

<Image
  data-src="https://source.unsplash.com/random"
  width={500}
  height={500}
  style={{ aspectRatio: 16 / 9 }}
/>

## API

| Props  | type | default | description              |
| ------ | ---- | ------- | ------------------------ |
| height |      |         | element height attribute |
| width  |      |         | element width attribute  |
