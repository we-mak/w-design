# Getting start

## Install

`@w-design/core` is the main core package of W-design kit, most of the required components are in this package.

You can start with

```bash

npm install --save @w-design/core styled-components
# or
yarn add @w-design/core styled-components
```

## Prerequisite: Provider component

All of the package need **`Provider`** to pass theme object to all the components,
so you must have `Provider` on the top of your App.

```jsx
import { Provider, Container } from "@w-design/core";

ReactDOM.render(
  <Provider>
    <Container> Lorem ipsum </Container>
  </Provider>,
  mountNode
);
```

Theme provider is based on [styled-components](https://www.styled-components.com/) and making theme with [styled-system](http://jxnblk.com/styled-system/)

Provider provides theme for your layout, you can supply your theme by insert your theme object by \`theme\` props

```js
<Provider>
  <App />
</Provider>
```

### Customize your theme

You can add custom them based on default theme

```jsx
<Provider
  theme={{
    fonts: "Roboto, Helvetica, sans-serif"
  }}
>
  <App />
</Provider>
```

### Extends with theme properties

You even can use you own theme with other component which is passed styled-system Api.

For example:

```jsx
<Container bg="N2">...</Container>
```
