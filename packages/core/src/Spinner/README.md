| Props       | type    | default | description          |
| ----------- | ------- | ------- | -------------------- |
| borderColor | string  |         | change spinner color |
| large       | boolean | false   | spinner large size   |

#Default

```js
<Spinner />
```

#Change color

```js
<Spinner borderColor="blue" />
```

#Large spinner

```js
<Spinner large />
```
