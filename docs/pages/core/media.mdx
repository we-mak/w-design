# Image

Image component with built-in lazy loading with `Intersection Observer`.

## Example

Fixed width image

<Image
  data-src="https://source.unsplash.com/random"
  width={320}
  height={500}
  style={{ border: "1px solid teal" }}
/>{" "}

```jsx
<Image
  data-src="https://source.unsplash.com/random"
  width={320}
  height={500}
  style={{ border: "1px solid teal" }}
/>
```

## API

| Props  | type | default | description              |
| ------ | ---- | ------- | ------------------------ |
| height |      |         | element height attribute |
| width  |      |         | element width attribute  |

# ImageRatio

## Example

<ImageRatio>
  <Image data-src="https://source.unsplash.com/random?building" />
</ImageRatio>

## API

| Props         | type                                                       | default | description                    |
| ------------- | ---------------------------------------------------------- | ------- | ------------------------------ |
| paddingBottom | number                                                     | 16/9    | aspect ratio by padding bottom |
| objectFit     | `"fill"`, `"contain"`, `"cover"`, `"none"`, `"scale-down"` |         | object-fit style attribute     |
| children      | React.ReactNode                                            |         | children node                  |
