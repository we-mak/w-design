import { Badge } from "@w-design/core";

# Badge

Badge is the number indicator. Mostly used as the quantity label of countable thing.

### Example

<Badge>{100}</Badge>
<Box m={10} />
<Badge appearance="primary">{100}</Badge>
<Box m={10} />
<Badge appearance="danger" maxCount={99}>
  {100}
</Badge>

```jsx
<Badge>{100}</Badge>
<Badge appearance="primary">{100}</Badge>
<Badge appearance="danger" maxCount={99}>
  {100}
</Badge>
```

### API

| Props      | Type                           | default   | description            |
| ---------- | ------------------------------ | --------- | ---------------------- |
| appearance | `default`, `primary`, `danger` | "default" | visual style of badge  |
| maxCount   | `number`                       |           | maximum count          |
| children   | `number`, `string`             |           | the value of the badge |
