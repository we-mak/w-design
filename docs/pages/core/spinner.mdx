# Spinner

Useful for pending render, loading data...

## Example

<Spinner />
<Spinner spinnerColor="green" />
<Spinner size="sm" />
<Spinner size="lg" />

```jsx
<Spinner />
<Spinner spinnerColor="green" />
<Spinner size="sm" />
<Spinner size="lg" />
```

## API

| Props       | type    | default | description          |
| ----------- | ------- | ------- | -------------------- |
| borderColor | string  |         | change spinner color |
| large       | boolean | false   | spinner large size   |
